# AI Student Management System

An intelligent student management platform powered by AI/ML capabilities for predictive analytics, personalized learning, and automated assessments.

## Features

### Core Features
- 📚 **Student Management**: Registration, profiles, enrollment tracking
- 📊 **Performance Tracking**: Grades, attendance, academic progress
- 🎯 **Course Management**: Course creation, scheduling, resources

### AI-Powered Features
- 🤖 **AI Chatbot**: Intelligent tutoring and query resolution
- 🔮 **Predictive Analytics**: Student performance prediction
- 📈 **Learning Path Recommendation**: Personalized course suggestions
- ⚠️ **Early Warning System**: Identify at-risk students
- 📝 **Automated Assessment**: AI-based assignment evaluation

### Admin Dashboard
- 📉 Analytics & insights
- 📋 Performance reports
- 👥 Student management
- 📚 Resource allocation

## Tech Stack

### Backend
- **Framework**: Node.js + Express.js
- **Database**: PostgreSQL
- **ORM**: Sequelize
- **Authentication**: JWT + bcrypt
- **AI/ML**: OpenAI API

### Frontend
- **Framework**: React 18
- **State Management**: Redux
- **UI Library**: Material-UI (MUI)
- **HTTP Client**: Axios
- **Styling**: Tailwind CSS

### DevOps
- **Containerization**: Docker & Docker Compose
